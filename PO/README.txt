Ce repository contient des solutions aux TP de PO de L2 de l'ISTIC. 
Je ne certifie pas l'absence d'éventuels bugs ou d'un code optimisé au mieux donc si vous constater un problème quelque part ou que vous souhaitez apporter votre contribution en proposant une autre version fonctionnelle du code, vous pouvez me contacter par le Discord. 
Je reste également disponible en cas de questions par rapport à ce dernier ou plus généralement au cours si vous n'avez pas compris l'une des notions (j'essayerais au mieux et si possible pour moi d'y répondre) !

Pour ouvrir directement le projet sur Eclipse :
- Copier cet URL : https://github.com/theogiraudet/PO.git
- Sur Eclipse, aller dans File/Import.../Git/Project/Clone URI
Normalement si l'URL a bien été copié, les champs devraient être pré-remplis.
- Cliquer ensuite sur Next jusqu'à pouvoir cliquer sur Finish

Le projet sera normalement importé sur votre PC !
